EcoShop - Sustainable Fashion E-Commerce Platform

A modern, eco-conscious e-commerce website built with React, TypeScript, and Tailwind CSS, showcasing sustainable fashion and lifestyle products.

Project Overview

EcoShop is a fully-featured e-commerce platform dedicated to sustainable fashion, offering organic, recycled, and environmentally-friendly clothing and accessories. The platform combines modern web technologies with a commitment to environmental responsibility.

Key Features

E-Commerce Functionality
- Product Catalog: Browse sustainable products with detailed descriptions, images, and specifications
- Shopping Cart: Add/remove items with real-time updates and quantity management
- Product Filtering: Filter by categories (T-Shirts, Hoodies, Pants, Accessories, Panjabi, Kids)
- Search & Discovery: Advanced search with sorting options (price, rating, popularity)
- Product Details: Comprehensive product pages with multiple images, reviews, and specifications
- Checkout Process: Secure multi-step checkout with payment integration
- Order Management: Complete order tracking and history

Modern UI/UX
- Responsive Design: Optimized for desktop, tablet, and mobile devices
- Dark Mode: Toggle between light and dark themes
- Smooth Animations: Engaging micro-interactions using Framer Motion
- Component Library: Built with shadcn/ui for consistent, accessible components
- Loading States: Skeleton loaders and smooth transitions

Sustainability Features
- Eco-Friendly Products: All items made from sustainable materials (organic cotton, hemp, bamboo, recycled materials)
- Carbon Neutral Shipping: Free eco-friendly shipping on orders over $75
- Product Transparency: Detailed sustainability information for each product
- Environmental Impact: Track plastic bottles saved and carbon offset

Technical Stack

Frontend
- React 18 - Modern React with hooks and concurrent features
- TypeScript - Type-safe development
- Vite - Fast build tool and development server
- Tailwind CSS - Utility-first CSS framework
- Framer Motion - Animation library
- React Router - Client-side routing
- React Query - Server state management
- Zustand - Client state management
- React Hook Form - Form handling with validation

UI Components
- shadcn/ui - Modern component library
- Radix UI - Accessible primitive components
- Lucide React - Icon library

Development Tools
